## Deployment

The site deploys automatically with GitHub Pages whenever changes are pushed to
the `main` branch.

## Custom Domain

When a domain is purchased, add it in GitHub under:

```text
Settings > Pages > Custom domain
```

Then create the DNS records GitHub shows for that domain. If the domain should
be committed to the repo, add a `CNAME` file containing only the domain name.
